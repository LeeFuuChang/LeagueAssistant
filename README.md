## 功能简介

💡	玩家查詢：「 對戰紀錄、戰棋紀錄、專精英雄、對局詳情、圖表化玩家數據 」

💡 	遊戲數據：「 OPGG英雄數據、OPGG戰棋數據、玩家挑戰數據 」

💡 	特殊功能：「 自動接受、秒選英雄、秒禁英雄、自動更換符文頁、自動更換召喚師技能 」

💡 	特殊功能：「 召喚師技能冷卻浮窗、遊戲中紀錄召喚師技能冷卻、一鍵發送召喚師技能冷卻 」
